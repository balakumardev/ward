A · Three-column source list — Mail / Finder idiom

Sidebar (harness switcher + categories + special views) → item list with security badges, grouped by scope → detail inspector/editor. The canonical macOS structure: dense, instantly familiar, best for browsing lots of config fast.

B · Sidebar + tabbed workspace — task-mode idiom

Left sidebar picks harness + scope; the main area is a set of top tabs (Organize / Security / Budget / Sessions / Backups), each getting the full width. Best when the security scan and budget views want room to breathe; detail slides in as a drawer.

C · Hybrid — modes in the sidebar — recommended

Top-level modes live in the sidebar (Organizer, Security, Budget, Sessions, Backups). Organizer gives you A's 3-column browsing; the heavier views (Security, Budget, Backups) take over the full width. Combines dense config browsing with room for the differentiating pillars.
